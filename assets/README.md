# Assets Folder

This folder contains app icons and splash screens.

## Required Files

- `icon.png` - App icon (1024x1024 recommended)
- `splash.png` - Splash screen image (1284x2778 recommended)

## Adding Your Logo

1. Create or resize your logo to 1024x1024 pixels
2. Save as `icon.png` in this folder
3. For the splash screen, create a 1284x2778 image with your logo centered
4. Save as `splash.png` in this folder

## Online Tools

You can use online tools to generate app icons:
- https://makeappicon.com/
- https://appicon.co/
- https://icon.kitchen/

## Temporary Placeholder

Until you add your own logo, the app will use Expo's default icon.
